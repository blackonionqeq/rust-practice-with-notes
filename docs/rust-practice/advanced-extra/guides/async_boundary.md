# Async Boundary Guide

Advanced-extra async exercises focus on application-level async Rust.

## In Scope

- `async fn`
- `.await`
- Tokio runtime setup
- Async file IO
- `join!` / `try_join!`
- Timeout handling

## Out of Scope

- Hand-written `Future` implementations
- `Waker`
- Executor internals
- Direct `Pin` usage
- Self-referential futures
- Unsafe async abstractions

Those topics belong to the expert section.
