## Topic Order

Known old-to-new basic topic order from the notes:

1. 命令行 / Cargo
2. 语法和理解 / 所有权
3. 字符串
4. 元组
5. 结构体
6. 枚举 / Option
7. Option 组合器
8. 数组
9. 流程控制
10. 模式匹配
11. 方法
12. 泛型和 trait
13. 错误处理
14. 生命周期
15. crate / package / module
