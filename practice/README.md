# Rust Practice Projects

Runnable Cargo projects live here.

## Layout

- `basic/`: basic exercises that match `docs/rust-practice/basic/`
- `advanced/`: advanced exercises and stage projects that match `docs/rust-practice/advanced/`
- `expert/`: expert exercises and stage projects that match `docs/rust-practice/expert/`

## Creating Projects

Create new basic projects with:

```bash
cargo new practice/basic/<number_name> --name <valid_package_name>
```

Create new advanced projects with:

```bash
cargo new practice/advanced/<number_name> --name <valid_package_name>
```

Create new expert projects with:

```bash
cargo new practice/expert/<number_name> --name <valid_package_name>
```

Cargo package names must not start with digits.
